## 案例二：自助法实验

### 1. 实验背景与目的

**自助法**（Bootstrap）是机器学习和统计学中一种重要的重采样技术。其核心思想是通过**有放回抽样**（Sampling with replacement）从现有数据集中生成多个与原始数据集规模相同的“自助样本集”，进而估计总体的统计量（如均值、方差等）及其分布情况。

本实验旨在通过代码完整模拟自助法的全过程，并通过统计学指标与可视化手段，直观分析自助样本集的统计特性及其收敛情况。

### 2. 实验流程设计

本代码实验主要分为以下核心步骤：

1. **数据生成**：随机生成 $N$ 个服从标准正态分布 $\mathcal{N}(0,1)$ 的随机数，作为初始数据集（原始样本）。
    
2. **自助采样**：从上述 $N$ 个样本中，执行有放回抽样 $M$ 次；每次抽取 $N$ 个数，最终构成 $M$ 个相互独立的自助样本集。
    
3. **统计计算**：分别计算这 $M$ 个自助样本集的均值与方差。
    
4. **可视化与分析**：对生成的统计量分布和收敛性进行汇总与图表绘制。
    

### 3. 实验参数与原始数据

为保证实验可复现，代码中固定了随机种子（`np.random.seed(42)`），具体参数与原始样本特征如下：

- **实验参数**：
    
    - 原始样本量 $N = 10$
        
    - 自助抽样次数 $M = 1000$
        
- **原始样本统计特征**：
    
    - **样本数据 $x$**：包含 $\{0.4967, -0.1383, 0.6477, 1.5230, -0.2342, -0.2341, 1.5792, 0.7674, -0.4695, 0.5426\}$
        
    - **样本均值**：$0.448061$
        
    - **样本方差**：$0.470467$
        
    - **样本标准差**：$0.6859$
        

### 4. 实验结果与统计分析

经过 $M=1000$ 次有放回抽样后，我们计算了这1000个样本集的均值和方差，统计汇总如下：

#### 4.1 均值（Means）的分布特征

- **均值的均值**：`0.448378`（极其接近原始样本的均值 `0.448061`）
    
- **均值的方差**：`0.044860`
    
- **均值的标准差**：`0.211801`
    
- **对比分析**：原始生成的 $N=10$ 样本存在一定随机偏差（其均值为0.448而不是理论的0），但通过 Bootstrap 计算出的均值期望非常完美地逼近了这10个原始样本的客观均值。
    

#### 4.2 方差（Variances）的分布特征

- **方差的均值**：`0.422101`
    
- **方差的方差**：`0.020085`
    
- **方差的标准差**：`0.141721`
    
- **对比分析**：自助法算出的平均方差略低于原始样本的方差（`0.470467`），这一现象在统计学上是合理的——因为“有放回抽样”会导致部分样本在单个子集中重复出现，从而使单次采样集内的“多样性”降低，方差呈现出一定的下偏估计。
    

### 5. 可视化分析结论

代码中生成了一张包含四个子图的综合大图（`Bootstrap Method Experiment`），它生动地展示了以下几个维度：

1. **原始样本分布（Step 1: Original Sample）**：展示了初始10个散点在标准正态分布密度曲线周围的落点情况，并标示了实际的初始均值。
    
2. **Bootstrap 均值分布（Bootstrap Means Distribution）**：1000个自助样本的均值在直方图中呈现出典型的正态分布（钟形曲线）规律，且拟合的高斯曲线中轴线精确对齐原始均值。
    
3. **Bootstrap 方差分布（Bootstrap Variances Distribution）**：同样对1000个方差数据进行了正态拟合与密度直方图的绘制。
    
4. **估计量的收敛曲线（Convergence of Bootstrap Estimates）**：绘图显示，随着横坐标（Bootstrap 重采样次数）的不断增加，累计均值曲线与累计方差曲线迅速平稳，消除了初期的震荡。这证明了大采样次数 $M$（如1000次）能够保证估计结果的可靠性。
    

### 6. 实验总结

通过本次代码实验，我们以程序化的手段证实了自助法的原理：**在极其有限的观测样本量下（本实验仅为 $N=10$），可以通过大规模的重采样，高效且稳定地模拟出样本特征分布的形态与边界**。实验既展现了自助法在无参数分布估计上的优势，也观察到了因初始数据分布变化而引入的微小估计偏差现象。这种思想在集成学习（如随机森林及其 OOB 验证）和区间估计中具有重要的实践价值。
